## Development

To run locally:

1. `git clone` the project
2. `cd pentominoes`
3. `npm i`
4. `npm run dev`

The tests are using `vitest`. If `npm run test` doesn't work then try updating your node/npm version; you'll have to delete `node-modules` and then rerun `npm i` for this to take effect.
